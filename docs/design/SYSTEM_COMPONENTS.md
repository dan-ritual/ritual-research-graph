# System Components

## Buttons
- **Family:** Mono
- **Transform:** Uppercase
- **Spacing:** 0.1em
- **Primary:**
  - Background: `var(--mode-accent)`
  - Text: White
- **Secondary:**
  - Border: `var(--border)`
  - Text: `var(--foreground)`
  - Hover: `var(--muted)`

## Cards
- **Background:** `var(--card)`
- **Border:** 1px solid `var(--border)`
- **Radius:** 0px (Sharp edges)
- **Shadow:** None

## Inputs
- **Background:** Transparent or White
- **Border:** 1px solid `var(--foreground)` (opacity 0.5)
- **Radius:** 0px
- **Font:** Mono, Uppercase, Caption size
- **Placeholder:** `var(--muted-foreground)`

## Badges
- **Font:** Mono
- **Transform:** Uppercase
- **Variant (Status):**
  - Border: Dotted 1px
  - Text: Semantic color (Green/Yellow/Red)

## Section Headers
- **Font:** Mono
- **Transform:** Uppercase
- **Tracking:** 0.12em
- **Color:** `var(--mode-accent)`
- **Border:** Bottom dotted 1px `var(--mode-accent)` (opacity 0.3)

## Navigation
- **Font:** Mono
- **Structure:** Simple list
- **Active State:** `var(--mode-accent)` text color
