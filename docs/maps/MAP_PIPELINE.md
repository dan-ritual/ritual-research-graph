        P->>B: Twitter thread search
        B-->>P: bird_output.json
    end
    P->>C: Synthesize all research
    C-->>P: Narrative research.md

    Note over P,C: STAGE 3: Entities (Claude)
    P->>C: Extract entities from all artifacts
    C-->>P: entities.json
    P->>DB: Check for duplicates
    DB-->>P: Existing entities
    P->>DB: Insert new entities

    Note over P,C: STAGE 4: Config (Claude)
    P->>C: Generate SITE_CONFIG
    C-->>P: SITE_CONFIG.js

    Note over P: STAGE 5: Build (Vite)

    P->>P: Inject SITE_CONFIG
    P->>P: npm install && vite build

    DB-->>P: Success

    P-->>U: Generation complete

flowchart TD
